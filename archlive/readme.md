# What is Indra?

Indra is a Distro Based on Arch Linux, with the two verticals being
* Desktop @ [Devendra][devendra.indra.org]
* Server  @ [Servendra][servendra.indra.org]


What does Devendra contain?

* Hyprland
* ly-dm
* custom bash scripts for use


What does Servendra container?

* customized zsh functions and aliases
* command line utilities to monitor and handle containers out of the box
